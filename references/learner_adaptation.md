# Learner Adaptation

## Principle

Teach to the student's current level. A correct technical answer can still fail if the student cannot understand the terms.

## Levels

### L0 Beginner

Profile: Does not know research questions, variables, mediators, moderators, or literature review logic.

Style:
- Use everyday language.
- Use one concept per paragraph.
- Avoid abbreviations unless explained.
- Give concrete examples.

Say:
- "中间原因" before "中介变量"
- "让影响变强或变弱的条件" before "调节变量"
- "前人还没解释清楚的地方" before "研究缺口"

Avoid:
- SEM, PROCESS, construct validity, moderated mediation, endogeneity

### L1 Early Learner

Profile: Knows the proposal structure but cannot connect literature, gap, variables, and methods.

Style:
- Plain language plus academic term.
- Use small templates.
- Give next tasks.

Focus:
- Topic compression
- Literature extraction
- Problem awareness
- Simple X -> Y and why/when logic

### L2 Operator

Profile: Understands independent/dependent variables and maybe mediators/moderators, but model logic may be messy.

Style:
- Use terms directly, then check them.
- Focus on role clarity, level consistency, measurement, and causal order.

Focus:
- Mediator vs moderator
- Parallel paths
- Method matching
- Measurement feasibility

### L3 Advanced

Profile: Can read literature and discuss theory/methods.

Style:
- Use technical terms.
- Challenge assumptions.
- Push theory contribution and construct clarity.

Focus:
- Theory contribution
- Boundary conditions
- Construct distinction
- Method limitations and causal identification

## Term Translation Rules

Use these explanations before or with terms for L0/L1:

- Research gap: "前人还没解释清楚的地方"
- Problem awareness: "为什么这不只是有趣，而是值得研究"
- Independent variable: "可能的原因或出发点"
- Dependent variable: "你想解释的结果"
- Mediator: "解释 A 为什么影响 B 的中间原因"
- Moderator: "让 A 的影响变强、变弱或改变方向的条件"
- Control variable: "不是主角，但需要避免它干扰结果的因素"
- Theory: "解释这些变量为什么能连起来的逻辑"
- Method: "用什么证据回答这个问题"

## Output Adaptation

L0 output sections:
- "你现在真正卡住的地方"
- "用人话说"
- "研究里这叫什么"
- "一个小例子"
- "下一步只做这件事"

L1 output sections:
- "当前卡点"
- "白话解释"
- "对应的研究术语"
- "可以怎么整理"
- "下一步任务"

L2/L3 output sections:
- "教学画像"
- "模型诊断"
- "构念与层级问题"
- "缺口类型"
- "方法适配"
- "风险"
- "修改任务"

## Golden Output Examples

### Scenario A: Beginner Topic Exploration

User: "我想研究 IP 联名，但是不知道怎么写。"

L0/L1 response pattern:

```text
你现在不是写作问题，而是题目还太像一个兴趣。

白话说：你说的是“我对 IP 联名感兴趣”，但研究计划书要回答“IP 联名里的什么现象，为什么值得研究”。

研究里这一步叫把主题变成研究问题。

我们先不讲复杂模型。下一步你只需要回答三件事：
1. 你想解释购买意愿、品牌好感，还是社交传播？
2. 你想看哪个 IP 或哪类产品？
3. 你觉得最奇怪的现象是什么？
```

### Scenario B: Intermediate Draft Review

User: "我写了品牌契合度和稀缺性都会影响购买意愿。"

L1/L2 response pattern:

```text
你现在已经有两个可能原因：品牌契合度和稀缺性。

最弱的地方是：你还没有解释它们为什么值得放在同一个模型里比较。

白话说：契合度像是在问“这个联名合不合理”，稀缺性像是在问“这个东西错过会不会后悔”。它们不是同一类原因。

学术上可以改成两条路径：
1. 契合度 -> 情感认同 -> 购买意愿
2. 稀缺性 -> FOMO/紧迫感 -> 购买意愿

下一步：找 2 篇契合度文献和 2 篇稀缺性文献，分别提取它们解释的机制。
```

### Scenario C: Advanced Model Check

User: "我想把弱连接作为中介变量。"

L2/L3 response pattern:

```text
这里要先检查构念层级。

弱连接原本是人与人之间的社会关系强度，不等于个体心理里的情感依附或 FOMO。

所以你需要决定弱连接在模型里的位置：
1. 作为情境：信息来自弱关系网友还是强关系朋友
2. 作为机制：稀缺商品引发分享/交换，从而形成弱连接扩散
3. 作为结果：联名活动后粉丝之间产生新连接

如果只是普通问卷，我建议不要直接把弱连接当中介。更稳妥的是把 FOMO 或社会展示价值作为心理机制，把弱连接放在理论背景或调节情境里。
```
