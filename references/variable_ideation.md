# Variable Ideation Rules

## Purpose

Help students propose variables from literature-based reasoning, not keyword brainstorming.

This file is a variable admission system, not a variable inspiration list.

## Role Decision Tree

- Ask "why/how" -> candidate mediator
- Ask "when/for whom/under what condition" -> candidate moderator
- Ask "what is the missing construct" -> candidate new construct
- Ask "which theory explains better" -> competing paths or parallel mediators

## Core Definitions

Mediator: The middle reason explaining why X affects Y.

Moderator: The condition that changes the strength or direction of X's effect on Y.

Control variable: A factor not central to the theory but included to reduce alternative explanations.

New construct: A clearly defined concept with boundaries, theory role, and measurement possibility.

## Literature-Based Variable Sources

- Future research sections
- Limitations sections
- Contradictory findings
- Theory transfer across contexts
- Unexplained mechanisms
- Boundary conditions
- Measurement gaps
- Practical phenomenon not explained by current theory

## Five Paths For Suggesting Variables

### Black-Box Path

Known X -> Y, unclear why.

Suggest a mediator only if literature explains X -> M and M -> Y.

### Boundary Path

Known X -> Y, unclear when or for whom.

Suggest a moderator only if theory predicts how the effect changes.

### Competing-Mechanism Path

Different theories explain Y through different logics.

Use parallel or competing paths, but keep master's proposals simple.

### Context-Transfer Path

Theory from context A is moved to context B.

Clarify whether constructs need redefining in the new context.

### Anomaly Path

Prior findings conflict.

First inspect definitions and methods; then propose a boundary condition.

## Anti-Stacking Rules

- One variable, one job.
- One model, one main storyline.
- Do not mix psychological states, behaviors, and network outcomes without a chain.
- Do not rename an existing construct as "new."
- Prefer mature constructs unless construct development is the research goal.
- If it cannot be measured, observed, or manipulated, keep it as theoretical discussion.

## Candidate Variable Output Template

- Variable name
- Role in model
- Plain-language meaning
- Academic definition
- Source literature
- Why it belongs here
- Closest competing constructs
- Measurement option
- Risk or limitation
